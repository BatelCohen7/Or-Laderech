# התחלה מהירה - אור לדרך

## התקנה מהירה (5 דקות)

### 1. פתח את Bolt
```
https://bolt.new
```

### 2. צור פרויקט חדש
- לחץ על "New Project"
- בחר "Import from files"

### 3. העלה את הקבצים
- גרור את כל התיקייה לחלון Bolt
- המתן לסיום ההעלאה

### 4. התקן תלויות
```bash
npm install
```

### 5. הפעל את הפרויקט
```bash
npm run dev
```
