# Or LaDerech – UI / UX Style Guide

## Product Values
- Trust & Legitimacy (אמון, רשמיות)
- Clarity & Accessibility (בהירות ונגישות)
- Community & Urban Renewal (קהילה, התחדשות עירונית)
- Simplicity over cleverness

Target users:
- Residents aged 45–80
- Low to medium technical literacy
- Multilingual audience

---

## Color Palette

Primary:
- Dark Blue: #0B2A4A (headers, primary buttons)
- Light Blue: #3B82F6 (CTAs, highlights)
- White: #FFFFFF (backgrounds)

Secondary:
- Gray Text: #475569
- Border Gray: #E5E7EB
- Success: #16A34A
- Warning: #F59E0B
- Error: #DC2626

Rules:
- No low-contrast text
- No light gray text on white
- Buttons always solid or outlined clearly

---

## Typography

Base font:
- Sans-serif, highly readable (e.g. Inter, Assistant, Rubik)

Font sizes:
- Page title: 28–32px
- Section title: 20–22px
- Body text: 16–18px
- Buttons: min 16px

Rules:
- Never smaller than 16px
- Line-height ≥ 1.5
- Avoid italics

---

## Layout Principles

- One primary action per screen
- Content grouped in Cards
- Generous spacing
- No dense tables for residents
- Important actions always visible without scrolling

---

## Buttons

Primary Button:
- Solid Light Blue
- Large padding
- Clear action verb

Secondary Button:
- Outline Dark Blue
- Used for cancel / back

Destructive:
- Red outline + confirmation modal

---

## Cards

Used for:
- Documents
- Votes
- Messages

Card includes:
- Title
- Status badge
- Short description (1 line)
- One primary action button

---

## Status Badges

- Pending: Yellow
- Signed / Completed: Green
- Closed / Expired: Gray
- Action required: Blue

Text always included (not color-only).

---

## Accessibility

- Keyboard navigable
- Buttons min height: 44px
- No color-only meaning
- Clear confirmation messages
- Explicit success / error feedback

---

## Language & Direction

Supported:
- Hebrew (RTL)
- Arabic (RTL)
- Russian (LTR)
- English (LTR)

Rules:
- Full RTL support (dir="rtl")
- Icons flip in RTL
- No hardcoded strings
- All text via translation keys

---

## Background Imagery

- Urban buildings
- Soft overlays (blue gradient)
- Never behind body text
- Used mainly on:
  - Landing
  - Login
  - Empty states

---

## Tone of Voice

- Respectful
- Simple
- Direct
- No jargon
- No slang
