# מדריך להקלטת סרטון הדגמה - אור לדרך

## הכנות לפני ההקלטה

### 1. הכנת הסביבה
- **דפדפן**: השתמש בדפדפן נקי (מומלץ Chrome) במצב חלון פרטי
- **רזולוציה**: הגדר את רזולוציית המסך ל-1920x1080
- **תצוגה**: הגדל את הדפדפן למסך מלא (F11)
- **ניקיון**: סגור כל חלונות או התראות אחרים שעלולים להפריע

### 2. תוכנת הקלטה
- **מומלצת**: OBS Studio (חינמית) או Camtasia (בתשלום)
- **הגדרות**: 
  - רזולוציה: 1920x1080
  - קצב פריימים: 60fps
  - פורמט: MP4
  - איכות: גבוהה

### 3. חשבונות לדוגמה
הכן מראש את פרטי ההתחברות הבאים:

- מספר ת.ז: 123456789

**מנהל**:

## תהליך ההקלטה

### 1. התחל את ההקלטה
- התחל את תוכנת ההקלטה
- ודא שאתה מקליט את החלון הנכון
- התחל את ההקלטה

### 2. עקוב אחר התסריט
עקוב אחר התסריט המפורט בקובץ `INVESTOR_DEMO_SCRIPT.md`:
1. כניסה לאתר והצגת דף הבית
2. הרשמה והתחברות של דייר
3. סיור בדשבורד הדייר
4. התחברות כמנהל
5. סיור בדשבורד המנהל
6. סיכום

### 3. טיפים להקלטה מוצלחת
- **מהירות**: שמור על קצב מהיר אך ברור
- **עכבר**: הזז את העכבר בצורה חלקה ומכוונת
- **הדגשה**: השתהה מעט על אלמנטים חשובים
- **שגיאות**: אם טעית, התחל מחדש את הסצנה הנוכחית
- **חלקות**: הימנע מתנועות פתאומיות של העכבר

## עריכה לאחר ההקלטה

### 1. חיתוך וקיצור
- חתוך כל השתהויות או שגיאות
- קצר את הסרטון ל-1:30 עד 2:00 דקות
- הוסף מעברים חלקים בין סצנות

### 2. הוספת טקסט (אופציונלי)
- הוסף כותרות קצרות לכל חלק
- הוסף הסברים קצרים לפיצ'רים מרכזיים
- שמור על פונט נקי וקריא (מומלץ Assistant או Alef)

### 3. הוספת מוזיקת רקע (אופציונלי)
- בחר מוזיקה שקטה ומקצועית
- שמור על עוצמה נמוכה שלא תסיח את הדעת
- ודא שהמוזיקה חופשית מזכויות יוצרים

### 4. ייצוא הסרטון הסופי
- פורמט: MP4
- קודק: H.264
- רזולוציה: 1920x1080
- קצב פריימים: 30fps או 60fps
- קצב סיביות: 8-10 Mbps
