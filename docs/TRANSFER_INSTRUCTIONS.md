# הוראות העברת פרויקט "אור לדרך"

## מה כלול בחבילה זו

הפרויקט כולל:
- ✅ אתר React מלא עם TypeScript
- ✅ עיצוב מתקדם עם Tailwind CSS
- ✅ דשבורד דיירים מקיף
- ✅ דשבורד מנהל מערכת
- ✅ מערכת אימות
- ✅ ניתוח זכויות תכנוניות
- ✅ מודלים תלת-ממדיים
- ✅ תמיכה ב-3 שפות (עברית, אנגלית, רוסית)
- ✅ נגישות מתקדמת
- ✅ עיצוב רספונסיבי

## הוראות התקנה למשתמש החדש

### שלב 1: יצירת פרויקט חדש ב-Bolt
1. היכנס ל-[bolt.new](https://bolt.new)
2. צור פרויקט חדש
3. בחר "Start from scratch" או "Import project"

### שלב 2: העלאת הקבצים
1. העלה את כל הקבצים מהתיקייה הזו
2. שמור על מבנה התיקיות כפי שהוא
3. וודא שכל הקבצים הועלו בהצלחה

### שלב 3: התקנת התלויות
```bash
npm install
```

### שלב 4: הפעלת הפרויקט
```bash
npm run dev
```

### שלב 5: פריסה (אופציונלי)
הפרויקט מוכן לפריסה ב-Netlify:
```bash
npm run build
```
