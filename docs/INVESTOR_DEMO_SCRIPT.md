# תסריט הדגמה למשקיעים - אור לדרך

## מטרת הסרטון
הדגמה קצרה (1-2 דקות) של פלטפורמת "אור לדרך" להתחדשות עירונית, המציגה את הפונקציונליות העיקרית עבור דיירים ומנהלים.

## תסריט הדגמה

### 1. כניסה לאתר (0:00-0:10)
- התחל בדף הבית של האתר
- הצג את העיצוב הנקי והמקצועי
- הדגש את הכותרת הראשית "אור לדרך - עתיד ההתחדשות העירונית"
- הצג את הפרויקט המרכזי "פרויקט צפת - שכונת כנען"

### 2. הרשמה והתחברות של דייר (0:10-0:30)
- לחץ על כפתור "התחבר עכשיו" בדף הבית
- עבור לדף ההרשמה לדיירים
- מלא את הטופס עם נתונים לדוגמה:
  - שם מלא: ישראל ישראלי
  - מספר ת.ז: 123456789
  - טלפון: 050-1234567
  - אימייל: israel@example.com
  - בחר בניין: זלמן שזר 1
  - מספר דירה: 15
  - סיסמה: Password1
- לחץ על "הרשם לפרויקט"
- הצג את הודעת ההצלחה
- עבור לדף ההתחברות
- התחבר עם מספר ת.ז וסיסמה

### 3. דשבורד דייר (0:30-0:50)
- הצג את דשבורד הדיירים המתקדם
- עבור בין הלשוניות השונות:
  - סקירה כללית - הצג את התקדמות הפרויקט, סטטיסטיקות ופעילות אחרונה
  - ציר זמן הפרויקט - הצג את אבני הדרך של הפרויקט
  - מערכת הצבעות - הצג את ההצבעות הפעילות
  - מרכז תקשורת - הצג את ההודעות והתקשורת עם היזם
  - מסמכים - הצג את המסמכים הזמינים
  - מודלים תלת-ממדיים - הצג את המודלים התלת-ממדיים של הפרויקט
- הדגש את הממשק הנוח למשתמש והעיצוב המודרני

### 4. התחברות כמנהל (0:50-1:10)
- התנתק מחשבון הדייר
- חזור לדף ההתחברות
- התחבר עם פרטי מנהל:
  - אימייל: oravraham217@gmail.com
  - סיסמה: Admin123
- הצג את המעבר האוטומטי לדשבורד המנהל

### 5. דשבורד מנהל (1:10-1:40)
- הצג את דשבורד המנהל המקיף
- עבור בין הלשוניות השונות:
  - סקירה כללית - הצג את הסטטיסטיקות הכלליות של המערכת
  - ניהול פרויקטים - הצג את רשימת הפרויקטים
  - ניהול דיירים - הצג את רשימת הדיירים הרשומים
  - ניהול מסמכים - הצג את המסמכים שהועלו
  - מרכז הצבעות - הצג את ההצבעות הפעילות
  - התנגדויות ובקשות - הצג את ההתנגדויות שהוגשו
  - מרכז ידע - הצג את התכנים הזמינים
- הדגש את היכולות הניהוליות המתקדמות

### 6. סיכום (1:40-2:00)
- חזור לדף הבית
- הדגש את היתרונות העיקריים של המערכת:
  - ממשק משתמש נוח ואינטואיטיבי
  - ניהול מתקדם של פרויקטי התחדשות עירונית
  - תקשורת יעילה בין דיירים, יזמים ובעלי מקצוע
  - שקיפות מלאה בכל שלבי הפרויקט
  - טכנולוגיה מתקדמת עם מודלים תלת-ממדיים

## טיפים להקלטה
1. **מהירות**: שמור על קצב מהיר אך ברור
2. **התמקדות**: הדגש את הפיצ'רים החשובים ביותר
3. **חלקות**: ודא שהמעברים בין המסכים חלקים
4. **איכות**: השתמש בהקלטת מסך באיכות גבוהה
5. **רקע**: הקלט בסביבה שקטה ללא הפרעות

## נקודות להדגשה למשקיעים
- **חדשנות**: הפלטפורמה הראשונה מסוגה בישראל
- **שוק**: פוטנציאל שוק של מיליארדי שקלים בתחום ההתחדשות העירונית
- **טכנולוגיה**: שימוש בטכנולוגיות מתקדמות כמו מודלים תלת-ממדיים
- **יעילות**: חיסכון של זמן וכסף בניהול פרויקטים
- **צמיחה**: פוטנציאל להתרחבות לערים נוספות ופרויקטים נוספים
